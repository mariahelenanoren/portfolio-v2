Cyberjump

A platform game written in Typescript with object-oriented programming (OOP) principles.

Cyberjump is a platform game set in Neo-Tokyo, 2076. Your objective is to stay alive for as long as you can while shooting as many aliens as you can. The code is written in typescript using OOP principles, we implemented the javascript library p5.js for canvas functionality. This was a group project by me, Oscar Andersson, Gustav Andersson, Lisa Pääjärvi and Sebastian Zazzi Jademyr.
